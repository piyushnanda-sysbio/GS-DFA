Steps to be followed:

1. Set the working directory to the folder in which the input datasets are stored:
    Use set.wd('Folder Location')
2. Run the code and the output file named 'MetTPMAll.csv' will be stored. 

N.B. The input dataset 'GSE147507' is obtained from GEO Omnibus in TSV format. The lengths of the isoforms in lung cells are obtained from Mammalian Transcriptomic Database. 

References:
